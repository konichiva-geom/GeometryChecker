PLAIN:
    Дан равнобедренный треугольник ABC, AB = BC.

    Докажите, что высота BH, медиана BM и биссектриса BD совпадают.

DESCRIPTION:
    new A; new B; new C
    bisector(new D, ABC)
    mid_point(new M, ABC)
    projection(new H, B, BC)

PROVE:
    M == D
    M == H

SOLUTION:
    equal_sided_triangles(AB == BC, BD == BD, ABD == CBD) => *
    adjacent_angle(ADB, BDC) => ADB + BDC == 180
    merge_projections(BH, BD, AC) => H == D
    merge_points_in_line(AM == AD, CM == CD) => M == D
